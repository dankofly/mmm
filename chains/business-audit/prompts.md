# Business Audit — Complete Prompts

## PROMPT 1 — Full Audit Request

```
Bewerte diese Business-Idee mit einem kompletten evidenzbasierten Audit. Keine Euphorie, nur harte Daten.

IDEE: [Beschreibe die Business-Idee hier]

Fuehre folgende Analysen durch:

1. MARKTANALYSE
- Direkte Wettbewerber (Produkte, Pricing, Nutzerzahlen, Status aktiv/tot)
- AI-Tools die dieselbe Leistung bieten (Preise, Geschwindigkeit)
- Kostenlose Alternativen (staatliche Programme, Open-Source, Freemium)
- Gescheiterte Unternehmen in diesem Space (Post-Mortems, Gruende)

2. UNIT ECONOMICS
- Customer Acquisition Cost (CAC) Berechnung
- Lifetime Value (LTV) Schaetzung
- Break-Even-Analyse
- Effektiver Stundenlohn bei Service-Modell

3. FRAMEWORK-TESTS
- DeMarco CENTS (Control, Entry, Need, Time, Scale) — jedes Ja/Nein
- Hormozi Value Equation (Outcome x Likelihood / Time x Effort)
- Thiel Zero-to-One Test (Monopol oder Kopie?)

4. 15-KRITERIEN SCORING (je 1-10)
- Problem-Schmerz
- Zahlungsbereitschaft
- Dringlichkeit
- Zugang zur Zielgruppe
- Vertriebshebel
- Skills-Passung
- Speed to MVP
- Speed to Revenue
- AI-Leverage (Moat oder Commodity?)
- Produktisierbarkeit
- Skalierbarkeit
- Blue Ocean vs Red Ocean
- Defensibility
- Personal-Brand-Fit
- Netzwerk-Fit

5. TIMING-ANALYSE
- Google Trends Richtung (steigend/fallend/stabil)
- Fliesst VC-Geld in den Space?
- Stellen Firmen in dem Bereich ein?

6. PRE-MORTEM
- Top 3 Szenarien die das Business in 12 Monaten toeten
- Antifragilitaets-Check (staerker oder schwaecher durch Schocks?)
- Plattform-Abhaengigkeiten

VERDICT: PURSUE / TEST / PARK / KILL mit klarer Begruendung.
```

## PROMPT 2 — Quick Validation (5 Minuten)

```
Schnell-Check fuer diese Idee in 60 Sekunden:

IDEE: [Kurzbeschreibung]

Beantworte nur:
1. CENTS-Check (5x Ja/Nein)
2. Wer zahlt heute schon fuer eine aehnliche Loesung?
3. Was ist der groesste Killer?
4. Score 1-10
5. Verdict: PURSUE / TEST / PARK / KILL
```

## PROMPT 3 — Competitor Deep Dive

```
Recherchiere alle direkten und indirekten Wettbewerber fuer:

IDEE: [Beschreibung]

Fuer jeden Wettbewerber:
- Name + URL
- Pricing
- Geschaetzte Nutzerzahl / Revenue
- Staerken
- Schwaechen
- Status (wachsend / stabil / schrumpfend / tot)

Dann: Wo ist die Luecke? Was machen alle schlecht? Wo ist der Blue Ocean?
```

## PROMPT 4 — Go-To-Market nach positivem Audit

```
Das Audit war positiv. Erstelle jetzt einen konkreten Go-To-Market Plan:

IDEE: [Beschreibung]
VERDICT: [TEST oder PURSUE]

Liefere:
1. ICP (Ideal Customer Profile) — wer genau, wo erreichbar
2. Erstes Angebot — was genau, welcher Preis, welches Format
3. Kanal — wo die ersten 10 Kunden herkommen
4. Outreach — exakte Nachrichten/Templates
5. Proof — wie erzeuge ich Glaubwuerdigkeit ohne Track Record
6. Timeline — was passiert in Woche 1, 2, 3, 4
7. Kill Condition — wann stoppe ich wenn es nicht funktioniert
```

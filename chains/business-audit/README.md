# Business Audit System

Evidence-based business idea validation. No euphoria, no guessing — hard data and frameworks.

**Created by:** Daniel Kofler / Council Framework
**Time:** ~20 minutes for a complete audit
**Output:** Scored verdict (PURSUE / TEST / PARK / KILL) with PDF report

## How It Works

6 steps run in parallel where possible:

| Step | What Happens | Duration |
|------|-------------|----------|
| 1. Research Agents | 6 parallel agents search the internet | ~5 min |
| 2. Skill Analysis | 8 specialized skills evaluate the idea | ~3 min |
| 3. Timing Analysis | Trends, sentiment, VC flows, job postings | ~3 min |
| 4. Framework Tests | CENTS, Hormozi, Thiel, 15-criteria matrix | ~2 min |
| 5. Founder-Fit + Pre-Mortem | Skill match, top-3 killers, antifragility | ~2 min |
| 6. Council Vote + Verdict | 10 filters evaluate, convergence check | ~3 min |

## Frameworks Used

- **CENTS** (DeMarco) — Control, Entry, Need, Time, Scale
- **Value Equation** (Hormozi) — Outcome x Likelihood / (Time x Effort)
- **0-to-1 Test** (Thiel) — Monopoly or copy?
- **15-Criteria Decision Logic** — Each scored 1-10
- **Build-Measure-Learn** (Ries) — Validated learning possible?
- **80/20** (Ferriss) — Which 20% creates 80% of value?
- **Cashflow Quadrant** (Kiyosaki) — E/S/B/I positioning

## Verdicts

| Score | Verdict | Meaning |
|-------|---------|---------|
| 8.0-10.0 | **PURSUE** | All signals positive. Start now. |
| 6.0-7.9 | **TEST** | Promising, validate 1-2 assumptions. |
| 4.0-5.9 | **PARK** | Interesting, wrong timing. |
| 1.0-3.9 | **KILL** | Fundamentally broken. Don't test. |

## Usage

Use the prompt from `prompts.md` to run a complete audit on any business idea.

# Viral Slides System

Create stunning viral presentations in ~5 minutes using Google NotebookLM.

**Source:** Sabrina Ramonov — "I Made a Viral PowerPoint in 5 Minutes"
**Tool:** Google NotebookLM (free)
**Output:** Shareable slide decks

## How It Works

1. Go to [notebooklm.google.com](https://notebooklm.google.com)
2. Add sources on the left (YouTube videos, web search, articles, documents)
3. Click the pencil icon in the **SLIDE DECK** section on the right
4. Paste a prompt from `prompts.md`
5. Click **GENERATE** (takes a few minutes)
6. Click the generated deck to open it
7. Use **START SLIDESHOW** for full-screen view

## Three Tiers

| Tier | Layouts | Features |
|------|---------|----------|
| Beginner | Basic | 4 design principles, clean output |
| Intermediate | 5 named | + visual metaphors, data viz, content structure |
| Advanced | 14 named | + typography rules, theme specs, safety rules |

## Tips

- **Custom outline:** Add your exact outline to the prompt if you want control over content
- **Brand match:** Update Theme and Typography sections for your brand
- **Share:** Click SHARE → "Anyone with a link" → Copy link
- **Iterate:** Try all 3 tiers on the same sources to see evolution

## Philosophy

> "Viral decks are not visually loud. They are intellectually loud. Say something new, interesting, and emotionally compelling."

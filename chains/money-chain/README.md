# Money Prompt Chain

A 5-prompt chain that transforms your skills into paying customers in a single AI conversation.

**Source:** Sabrina Ramonov — "I Asked ChatGPT To Make Me As Much Money As Possible"
**Time:** ~10 minutes
**Works with:** ChatGPT, Claude, DeepSeek

## Rules

1. **ALL prompts run in the SAME conversation** — do not start new chats between prompts
2. **Have a dialogue after each prompt** — push back, question, propose alternatives
3. **Don't accept the first answer passively** — the chain improves through your interaction
4. **Speed to first dollar** — the goal is your first paying customer, not a perfect plan

## Chain Flow

```
PREREQUISITE: Feed context about yourself
        |
        v
PROMPT 1: Identify 1 offer + 1 channel (then dialogue)
        |
        v
PROMPT 2: Identify #1 mentor to learn from
        |
        v
PROMPT 3: AI role-plays as mentor → 90-day plan (first customer ASAP)
        |
        v
PROMPT 4 (Intermediate): Hormozi validation → 3 tests under $50
        |
        v
PROMPT 5 (Advanced): AI coaches YOU → diagnose block → 2-week sprint
```

## Usage

Copy each prompt from `prompts.md` in sequence. Read the instructions between prompts carefully.

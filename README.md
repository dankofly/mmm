# MMM - Money Making Machine

AI-powered prompt chains and systems for generating revenue, validating business ideas, and creating viral content.

Based on proven frameworks from Sabrina Ramonov, Alex Hormozi, and the Council methodology.

## Systems

### 1. Money Prompt Chain (`/chains/money-chain/`)
A 5-prompt chain that goes from "I have skills" to "I have paying customers" in one conversation.
- Identify your best offer + channel
- Find the right mentor model
- Generate a 90-day action plan (first-dollar-first)
- Validate with $50 tests
- Diagnose mindset blocks + 2-week proof sprint

### 2. Viral Slides System (`/chains/viral-slides/`)
NotebookLM prompt system for creating viral presentations in 5 minutes.
- 3 tiers: Beginner, Intermediate, Advanced
- 14 proven viral layouts
- 8-point content structure framework
- Typography + theme specifications

### 3. Business Audit System (`/chains/business-audit/`)
Evidence-based business idea validation with 15-criteria scoring.
- Parallel research agents
- Council framework (10 mental models)
- CENTS, Hormozi, Thiel tests
- Timing analysis + Pre-Mortem

## Quick Start

1. Pick a system from `/chains/`
2. Read the `README.md` in that folder
3. Copy prompts into your AI tool (ChatGPT, Claude, etc.)
4. Follow the chain sequence — don't skip steps
